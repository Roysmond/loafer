SpringData ES中一些底层原理的分析

 之前写过一篇SpringData ES 关于字段名和索引中的列名字不一致导致的查询问题，顺便深入学习下Spring Data Elasticsearch。 Spring Data Elasticsearch是Spring Data针对Elasticsearch的实现。 它跟Spring Data一样，提供了Repository接口，我们只需要定义一个新的接口并继承这个Repository接口，然后就可以注入这个新的接口使用了。 定义接口： @Repository public interface TaskRepository extends ElasticsearchRepository<Task, String>
